# PracticalQuiz1 — Makefile Automation & GitHub

## Project Structure
```
PracticalQuiz1_20260220/

├── src/
```

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
make preprocess     # Load, clean, and split dataset
make train          # Train Logistic Regression model
make evaluate       # Run evaluation and save results
make all            # Run full pipeline
make clean          # Remove generated files
```
